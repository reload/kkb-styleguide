Kalender

Kalenderen kan vise lige så mange arrangementer pr dag som man ønsker. Man tilføjer blot indholdet i li.calender-weekday som er dage som f.eks. Mandag, Tirsdag etc.

For hver li.calender-weekday man opretter, skal der være et tilsvarende li element i .calendar-nav som er menuen.

Første element i li.calender-weekday svarer til første element i .calendar-nav menuen.

Kalender funktionen er afhængig af jQuery, calendar.js, Modernizr + css-calc.js (for IE8).
